2. 计算机网络性能指标

2.1 速率

数据传输率/比特率：主机在数字信道上传送数据位数的速率

1 Tb/s = 10^3 Gb/s = 10^6 Mb/s = 10^12 b/s

2.2 带宽

带宽表示网络的通信线路传送数据的能力, 指单位时间内从某一点到另一点的最高数据率

2.3 吞吐量

单位时间内通过某个网络(或信道、接口)的数据量

2.4 时延

数据从网络的一端传送到另一端所需要的时间亦称延迟/迟延

发送时延：分组发送的第一个比特到最后一个比特所需的时间 发送时延 = 数据长度/信道带宽(发送速率)

传播时延：取决于电磁波传播速度和链路长度 传播时延 = 信道长度/电磁波在信道上传播速率

排队时延：比特流在路由器或交换机前排队等候处理的时间

处理时延：路由器或交换机处理数据检错找出口的时间

2.5 时延带宽积

时延带宽积又称以比特为单位的链路长度 用来描述容量，即某段链路现在有多少比特

时延带宽积 = 传播时延 * 带宽

2.6 往返时延RTT

从发送方发送数据到发送方接受到接受方的确认所需要的时间

RTT = 传播时延*2 + 末端处理时间

2.7 利用率

信道利用率：有数据通过时间/数据通过时间

网络利用率：信道利用率加权平均值

网络利用率越高，时延越大

3. 计算机网络分层结构

  • 实体: 第n层中的活动元素称为n层实体，同一层实体称为对等实体

  • 协议：为对等实体建立的规则 [水平]

    ​ 语法：规定传输数据与控制信息所用的格式

    ​ 语义：规定所要完成的功能

    ​ 同步：规定各种操作的顺序

  • 接口：又称访问服务点SAP，上层使用下层服务的入口

  • 服务：下层为相邻上层提供的功能调用 [垂直]

SDU服务数据单元 (Service Data Unit): 能够完成用户所要求的功能而应传送的数据

PCI协议控制信息(Protocol Control Information): 控制协议操作的信息

PDU协议数据单元(Protocol Data Unit): 对等层次之间传送的数据单位

​ PDU = PCI + SDU

5. TCP/IP模型

应用层： HTTP、FTP、DNS

传输层: TCP、UDP

网际层: IP

网络接口层: Ethernet、ATM、Frame Relay

6. 5层参考模型

应用层： HTTP、FTP、DNS

传输层: TCP、UDP

网络层: IP

数据链路层：PPP、以太网

物理层：比特传输
