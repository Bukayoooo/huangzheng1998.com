1. 串的基本操作

串的定义

由零个或多个字符组成的有限序列

子串在主串中的位置：子串的第一个字符在主串中的位置

串是一种特殊的线性表，里面存放着char型字符。串的增删改查等基本操作通常以子串为对象

串的基本操作

T=”” S=”HuangZheng is a handsome boy” W=”Yes”

  • 赋值： StrAssign(&T, chars);
  • 复制： StrCopy(&T, S);
  • 判空： StrEmpty(S);
  • 串长： StrLength(S);
  • 清空： StrClean(S);
  • 销毁： StrDestroy(S);
  • 链接： StrConcat(&T, S1, S2);
  • 子串： SubString(&Sub, S, pos, len) 将S的第pos位起，len个长度的子串赋给Sub
  • 定位： Index(S, T) 若主串S中存在与串T相同的子串，返回第一次出现该子串的位置
  • 比较： StrCompare(S, T) if S>T, return >0; else <0;

3. 朴素模式匹配算法分析

设主串长为m, 模式串长为n; m>>n

长为m的主串共有m-n+1个长为n的模式串

  • 匹配成功

    最好时间复杂度：第一个模式串即成功 O(n)

    最坏时间复杂度：最后一个模式串匹配成功，且前面所有的模式串都已最坏的方式匹配失败 O((m-n+2)*n + n) = O(mn-n^2+3n) = O(mn)

  • 匹配失败

    最好时间复杂度：所有串都匹配失败，且都是在模式串第一个字符匹配失败 O(m-n+1) = O(m)

    最坏时间复杂度： 所有串都匹配失败，且都是在模式串最后一个字符匹配失败 O((m-n+1)*n) = O(mn)

综上所述，朴素模式匹配算法的最好时间复杂度是O(n)，最坏时间复杂度是O(mn)
